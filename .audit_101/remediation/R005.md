# R005 — Indexer Ponder config key drift (P0-HIGH)

**Source finding:** agent 096 — `indexer/ponder.config.ts` used legacy `chains`
key with `{ id, rpc }` entry shape and `chain:` per-contract; Ponder 0.8.30
expects `networks` with `{ chainId, transport }` and `network:` per-contract.
The mismatch made `createConfig`'s generic infer the registry as `never`,
which cascaded into 102+ TS errors in `indexer/src/index.ts` (every
`ponder.on(...)` call rejected because the event-name union was `never`).

## Reference

`indexer/node_modules/ponder/dist/index.d.ts`:

- L100-103: `type Config = { networks: { [networkName: string]: NetworkConfig$1<unknown>; }; ... }`
- L122-128: `createConfig` signature requires `networks:`
- L155-181: `NetworkConfig$1<network>` shape — `chainId: number; transport: Transport; ...`
- L189-197: `GetContractNetwork` — per-contract key is `network:` (string referencing a `networks` entry)

## Diff

`indexer/ponder.config.ts` lines 353-422.

```diff
 export default createConfig({
-  chains: {
+  // AUDIT R005: Ponder 0.8.x expects top-level `networks` (not `chains`) with
+  // entries shaped `{ chainId, transport }` (not `{ id, rpc }`). The drift
+  // resolved the registry to `never` and broke the typecheck across every
+  // handler in src/index.ts. See node_modules/ponder/dist/index.d.ts L100-181.
+  networks: {
     mainnet: {
-      id: 1,
-      rpc: rpcTransport,
+      chainId: 1,
+      transport: rpcTransport,
     },
   },
   contracts: {
     TegridyStaking: {
       abi: TegridyStakingAbi,
-      chain: "mainnet",
+      network: "mainnet",
       address: "0x626644523d34B84818df602c991B4a06789C4819",
       startBlock: 24500000,
     },
     // ... 8 more contract entries, each `chain: "mainnet"` → `network: "mainnet"`
   },
 });
```

Total token replacements:
- 1 × `chains:` → `networks:`
- 1 × `id:` → `chainId:`
- 1 × `rpc:` → `transport:`
- 9 × `chain: "mainnet",` → `network: "mainnet",` (one per contract:
  TegridyStaking, TegridyRestaking, RevenueDistributor, VoteIncentives,
  LPFarming, TegridyLending, SwapFeeRouter, CommunityGrants, MemeBountyBoard)

No contract addresses, ABIs, event subscriptions, start blocks, RPC fallback
list, or transport options were changed — purely a key-rename to match the
installed Ponder major's published config type.
