## Drift markers added

The reconciled tests carry inline `DRIFT (RC10)` comments noting which audit
finding was meant to be exercised by the original test, and what design point
was deferred. Tests carrying a `_DRIFT` or `_currentBehavior` suffix in the
function name pin the CURRENT (pre-fix) behavior so the test FAILS the moment
the deferred fix lands — a deliberate signal to update the assertion.
