# R035 — Frontend doc/address drift remediation

## Source findings (agent 055, HIGH)
Six concrete drifts where copy/pages contradicted on-chain or canonical truth:

1. `SecurityPage.tsx` hardcoded a paused v1 staking address (`0x65D8…0421`) and a stale `TegridyNFTLending` address (`0x63baD…68aD`) plus other addresses by hex literal — drifted from `frontend/src/lib/constants.ts`.
2. `TermsPage` §7 advertised a 0.3% swap fee — actual `SWAP_FEE_BPS = 50` in `SwapFeeRouter.sol` = **0.50%** (capped 1.00%).
3. `TokenomicsPage SUPPLY_DATA` showed a 65/20/10/5 pie — `TOKENOMICS.md` authoritative split is **30 LP / 10 Treasury / 10 Community / 5 Team / 45 Circulating** (0% Investors).
4. `FAQPage` Staking section claimed "1–52 month" lock, "1x→2.5x" boost, and "JBAC/Nakamigos/GNSS = 10–20% boost" — actual contract bounds are **7d–4y, 0.4×–4.0× linear, +0.5× JBAC-only** (4.5× ceiling via `MAX_BOOST_BPS_CEILING = 45000`).
5. `ContractsPage` `GITHUB_BASE` pointed at the (404) `tegridyfarms` org; trailing tracked-issues link used `tegriddy-farms` (typo, double-d). Source links are silently broken.
6. The repo source-of-truth is `https://github.com/fomotsar-commits/tegridy-farms.git` (single d) per `git remote -v`; that matches the SecurityPage / RisksPage links.

## Authoritative sources consulted
- `frontend/src/lib/constants.ts` — canonical contract addresses (Wave 0 redeploys included)

- `contracts/src/SwapFeeRouter.sol` — `feeBps` storage and timelocked setter confirm dynamic source
