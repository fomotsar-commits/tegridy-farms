# R041 — NFT image / metadata URI safety (frontend)

**Severity:** HIGH (agent 059)
**Status:** Patched. `tsc --noEmit` clean.

## Problems

- **H-1 XSS via attacker-controlled contractURI.** `useNFTDropV2.resolveContractUri`
  passed any URI through unchanged — `javascript:`, `file:`, `gopher:`, raw
  `data:image/svg+xml` (inline `<script>`) all flowed into `fetch()` and
  `<img src>`.
- **H-2 single-gateway IPFS.** `ipfs://` URIs hit one gateway with no fallback.
- **H-3 unbounded metadata fan-out.** `NftImage` fired one `/api/alchemy`
  request per grid cell on first paint (1000 cells → 1000 concurrent
  requests, 429 storms, $$$).
- **H-4 broken `<img>` + CLS.** `ArtImg.tsx` had no `onError`, no
  `width`/`height`, so a 404 art override rendered empty with layout jump.

## Fix

New module **`frontend/src/lib/imageSafety.ts`**:

- `ALLOWED_SCHEMES = ['https', 'ipfs', 'ar', 'data']` per OpenSea metadata
  standard. `data:` is restricted to a fixed MIME allowlist
  (`png|jpeg|webp|gif`) — `image/svg+xml` is **rejected outright** (safest
  sanitiser is "don't render SVG you didn't author"; mandate's DOMPurify
  path can be added behind a flag if a future surface needs it).
- `IPFS_GATEWAYS = [cloudflare-ipfs, dweb.link, ipfs.io]` round-robin.
  `fetchWithIpfsFallback` walks the list, returns the first 2xx, propagates
  AbortError immediately.
- `createLimit(n)` — tiny inline `p-limit` clone (no new top-level dep).
  `metadataLimit` = shared 5-wide gate.
- `PLACEHOLDER_NFT = '/placeholder-nft.svg'` (new same-origin asset).

Wired into:

- **`hooks/useNFTDropV2.ts`** — `resolveContractUri` / `resolveAssetUrl`
  now route through `resolveSafeUrl`; disallowed schemes return `''` /
  `undefined` so the page falls back to on-chain name/symbol. The
  `useEffect` guards with `isAllowedUri` before networking and uses
  `fetchWithIpfsFallback` for the round-robin.
- **`nakamigos/components/NftImage.jsx`** — both metadata-API fetches
  (initial-load and fallback-chain) wrapped in `metadataLimit(...)`. All
  resolved URLs pass through `safeUrl(...)` before assignment to `src`.
  Final `<img>` falls back to `PLACEHOLDER_NFT` if the safety gate rejects.
- **`components/ArtImg.tsx`** — added `onError` swap to `PLACEHOLDER_NFT`,
  default `width=1200 height=800` to reserve layout (caller props
  override). Caller-supplied `onError` is preserved.
